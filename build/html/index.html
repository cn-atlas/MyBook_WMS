Welcome to Python WEB 企业级实战’s documentation!

本手册通过带领读者从零开始搭建一个企业级 WMS（仓库管理系统），一步步剖析如何使用 Django + DRF 做一个前后端分离的企业级项目。本手册 源文件 和附带 配套源代码 可以在 Github 上找到。

备注

本手册配套源代码采用 GPL v3 开源协议，可以直接拿来使用哦！ 建议使用最新版本的软件包进行学习。
